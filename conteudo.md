## Operadores Lógicos
``` 
Positivo -> 1  
Negativo -> 0    
  
== : comparação positiva  
!= : comparação negativa  
!  : negacao  
```

## Operadores Relacionais
```
>  : maior que  
<  : menor que  
&& : and  
|| : or  
``` 

## Modificadores de Type
```
signed   -> + e -
unsigned -> +
long     -> aumentar a capacidade de armazenamento da nossa variável
short    -> nos queremos diminuir o tamanho de armazenamento
```
